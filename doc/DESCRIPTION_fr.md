## Baserow permet de créer des bases de données et des applications sans besoin de coder.  
La version fournie par défaut (**OSS**) est une version limitée.    
La version **complète** peut être installée: elle permet de trouver les fonctionnalités des versions Premium et Enterprise mais la plupart de ces fonctionnalités ne sont accessibles qu'**avec les formules d'abonnement payantes** et **n'ont pas été testées.**
