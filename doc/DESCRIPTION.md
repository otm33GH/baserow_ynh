## Baserow allows you to create databases and applications without the need to code.  
The shipped version is the **OSS version** with limitations. 
The **complete version** can be installed: it provides access to the features of the Premium and Enterprise versions, but most of these features are only accessible **with paid subscription plans** and **haven't been tested.**
The OSS version does not provide access to the features of the Premium and Enterprise versions. These features are only available with paid subscription plans.
